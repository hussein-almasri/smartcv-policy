# SmartCV Privacy Policy

Last updated: 2026

SmartCV respects your privacy. This Privacy Policy explains how we collect, use, and protect your information when using the SmartCV mobile application.

## Information We Collect

When using SmartCV, we may collect the following information:

- Email address (for authentication)
- Name (provided during account registration)
- Resume information entered by the user
- Profile image uploaded by the user

## How We Use Your Information

We use the collected information to:

- Create and manage user accounts
- Allow users to build and manage their resumes
- Improve application functionality
- Provide a personalized user experience

## Data Storage

User data is securely stored using Firebase services including:

- Firebase Authentication
- Cloud Firestore
- Firebase Storage

## Data Sharing

SmartCV does not sell, rent, or share user data with third parties.

## User Control

Users can edit or delete their information at any time through the application.

## Security

We take reasonable measures to protect your information from unauthorized access.

## Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be posted on this page.
